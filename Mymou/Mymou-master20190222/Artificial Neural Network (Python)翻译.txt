用于训练面部识别模块的Python代码

####准备数据（PrepareData.py）
数据应采用[（一个热阵列标签）格式（使用Mymou系统生成的图像阵列）]
例如。
[0,1,255,213,24,89，...]＃主题1
[1,0,39,210,130,198，..]＃主题2

这可以使用PrepareData.py脚本生成，该脚本在包含以下内容的文件夹中运行：
1个包含主题1的所有图像的文件夹
1个包含主题2的所有图像的文件夹
1个文件夹，包含与前2个文件夹中的所有图像对应的所有整数数组

然后脚本对所有整数数组进行排序，查找相应的.jpg是否在folder1或folder2中，并为数组添加适当的一个热标签

然后将所有这些数据堆叠起来并保存为一个大文件，该文件将与主要的人工神经网络脚本一起运行。


####训练神经网络（main.py）
使用PrepareData.py生成的示例数据运行

以下是：
	读入数据并拆分x和y数据
	标准化数据并保存Mymou系统用于在线图像识别的标准化参数
	将它们保存为meanAndVar.txt
	按照用户指定的比率将数据拆分为训练集和测试集
	使用用户指定的参数训练网络
	保存训练好的重量，然后Mymou系统可以使用这些重量进行在线图像识别
	wi.txt - 输入隐藏层的权重
	wo.txt - 隐藏输出层的权重

具有可由用户调整以改变网络性能的各种参数：
    varInputSize = 30552＃输入图层中的神经元（图像中每像素1个）
    varHiddenLayerSize = 20＃隐藏层中的神经元
    varOutputSize = 2＃输出层中的神经元（每个主题1个）
    varMomentum = 0.0001＃帮助避免局部最小值
    varIterations = 50＃通过模型运行所有训练数据的次数
    varRateDecay = 0.01＃通过训练迭代这个因子降低学习率
    varLearningRate = 0.01＃在每一步调整权重的数量
    varNumSubjects = 2＃科目数
    trainingToValidationRatio = 0.85＃训练集与验证集中保持的数据比率
